### E1

### E2
通过E1就可以知道找尽量少的非严格递增不相交子序列的数量，每一种子序列都用相同的颜色涂即可  
我们用f数组记录26个字母当前所涂的最大颜色值，即fi表示第i个字母当前所涂的最大值  
用color记录颜色  
遍历字符串  
假设当前遍历到的字母为c，则要找d~z里面的字母当前涂的颜色的最大值是多少，将当前字母c的颜色涂为最大值+1，因为这个c要交换到前面去，要与它之前出现的比它大的字母的颜色不用才行，然后不断地更新f数组
