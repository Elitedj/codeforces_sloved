### C
对每一个数进行因子分解，记录下该因子的幂次  
对每一个因子的幂次从小到大排序  
若某个因子出现次数小于n-1，则该因子在最后是没有贡献的  
因为至少两个数不包含该因子，那么他们的lcm肯定也不包含该因子，那么最后的gcd也不包含该因子  
若该因子出现n-1次，则贡献为pow（因子，最小幂次）  
若该因子出现n次，则贡献为pow（因子，次小幂次）
